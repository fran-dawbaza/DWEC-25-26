map

map → transformar cada elemento
- Sirve para transformar los elementos, devolviendo un nuevo array del mismo tamaño.

Ejemplo: obtener solo los nombres, crear un nuevo objeto con estado (aprobado / suspenso)

filter

filter → seleccionar algunos elementos
- Sirve para quedarnos solo con los elementos que cumplen una condición.

Ejemplo: alumnos aprobados, alumnos mayores de 18 años

reduce

reduce → acumular en un único valor
- Reduce el array a un único resultado (número, objeto, array, etc.).

Ejemplo: nota media del grupo, número de aprobados y suspensos

Podemos hacer combinaciones

Tanto map como filter retornan un nuevo array, que es suceptible de invocar de nuevo a otros métodos de los arrays

Ejemplo: nombres de alumnos aprobados, nota media de los aprobados, media de edad de los suspensos
